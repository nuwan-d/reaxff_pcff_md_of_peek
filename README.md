# Atomistic Models of Amorphous PEEK for LAMMPS (ReaxFF & PCFF)

This repository provides LAMMPS input scripts and data files for atomistic modeling of amorphous polyether ether ketone (PEEK) using ReaxFF and PCFF force fields.

The models are designed for molecular dynamics (MD) simulations of polymer deformation, damage evolution, and failure mechanisms at the nanoscale.

## 📦 Contents

- `reaxff/` – LAMMPS input files and data files for ReaxFF simulations  
- `pcff/` – LAMMPS input files and data files for PCFF simulations
